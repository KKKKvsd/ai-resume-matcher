# AI 开发岗位常见能力要求

AI 开发岗位通常关注以下能力：

## 技术栈
- Python
- FastAPI / Flask
- MySQL / PostgreSQL
- Redis
- Docker
- Git

## 大模型相关能力
- Prompt Engineering
- RAG（检索增强生成）
- Agent
- Tool Calling
- Memory
- 多轮对话系统
- OpenAI 兼容接口调用
- 向量数据库

## 工程能力
- API 设计与开发
- 模块化开发
- 日志与异常处理
- Docker 部署
- 性能优化
- 测试与调试

## 项目经历关注点
- 是否做过完整 AI 应用系统
- 是否接入过大模型接口
- 是否有 RAG / Agent 实践
- 是否有从 0 到 1 的项目落地经历
- 是否能把 AI 能力集成到实际业务中
