# 项目描述示例

## AI 应用项目描述示例
基于 FastAPI、MySQL 与 OpenAI 兼容接口开发 AI 简历分析与岗位匹配系统，实现用户注册登录、岗位管理、PDF 简历解析、匹配分析与历史结果查询，并设计大模型失败自动回退到 mock 分析的降级机制。

## RAG 项目描述示例
基于文档切分、向量检索与大模型生成能力，构建检索增强的问答系统，实现知识库检索、上下文增强与结构化回答输出，提升问答准确率与可解释性。

## 后端项目描述示例
基于 FastAPI/MySQL 实现后端服务，完成数据库建模、接口开发、JWT 鉴权、日志与异常处理，并通过 Docker 实现项目部署。
